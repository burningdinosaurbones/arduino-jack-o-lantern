# arduino-jack-o-lantern

This is an example of a publicly-visible repository hosting code and some description of hardware.

- pumpkin/pumpkin.ino is an Arduino sketch designed to flash red, white, and blue LEDs in a sequence.
- pumpkin/pumpkin_shield.fzz is a Fritzing design for a shield that implements the LEDs

Be sure to enclose the Arduino and battery in a plastic bag, so they don't get pumpkin juice all over them.
